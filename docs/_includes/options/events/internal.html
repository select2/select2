Internal Select2 events

Select2 has an internal event system that works independently of the DOM event system. This internal event system is only accesssible from plugins and adapters that are connected to Select2.

Select2:open

Fired after the select2 dropdown has opened. Example:

```javascript $('#example').on('select2:open', function (){ }) ```

Select2:opening

Fired before select2 dropdown opens. Example:

```javascript $('#example').on('select2:opening', function (){ }) ```

Select2:close

Fired after select2 dropdown has closed. Example:

```javascript $('#example').on('select2:close', function (){ }) ```

Select2:closing

Fired while select2 dropdown is closing. Example:

```javascript $('#example').on('select2:closing', function (){ }) ```

Select2:select

Fired after selection has been made. Example:

```javascript $('#example').on('select2:select', function (){ }) ```

Select2:selecting

Fired after a selection is made. Primary use is for Multiple selections. Example:

```javascript $('#example').on('select2:selecting', function (){ }) ```

Select2:unselect

Fired after a selection is deselected. Primary use is for Multiple selection scenarios. Example:

```javascript $('#example').on('select2:unselect', function (){ }) ```

Select2:unselecting

Fired while a selection is being deselected. Primary use is for Multiple selection scenarios. Example:

```javascript $('#example').on('select2:unselecting', function (){ }) ```
