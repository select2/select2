Can I force the dropdown positionning ?

You could choice between this options :

  • both : dropdown choice the best place between below and above (by default)
  • below : dropdown is always below
  • above : dropdown is always above
{% highlight js linenos %} $('select').select2({ dropdownPosition: 'above' }); {% endhighlight %}
